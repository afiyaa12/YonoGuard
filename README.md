# YONO-Guard 

YONO-Guard is an AI-powered cybersecurity solution designed to detect phishing links and prevent fraudulent mobile applications targeting SBI users.

## Features
- Phishing link detection using AI/ML
- APK integrity verification (SHA-256)
- UI and behavioral fraud detection
- Real-time warning system
- User awareness and nudge-based intervention
